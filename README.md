# 防线守卫 · 塔防游戏

浏览器中直接运行的完整多关卡塔防游戏（20 关 + 成就系统）。

## 如何开始

双击打开 `index.html`，或拖入 Chrome / Edge。

## 核心规则

### 自动波次
- **开局**：约 12–22 秒后自动开始第 1 波（可提前）
- **波次间隙**：清完当前波后约 **8–18 秒**自动开下一波（关卡越后间隔越短）
- **提前开波**：点击按钮或按 `空格`，根据剩余倒计时获得金币奖励

### 关卡解锁
- 默认仅解锁 **第 1 关**
- 通关第 N 关后解锁第 N+1 关
- **未解锁关卡不可进入**
- 已解锁关卡可在「选择关卡」中重玩

### 成就
主菜单「成就」查看全部 24 项成就。达成时右上角弹出提示，进度保存在本地。

## 操作

| 操作 | 说明 |
|------|------|
| 选塔 / 放置 | 底部选塔 → 点空地 |
| 升级 / 出售 | 点击已有塔 |
| 提前开波 | 按钮或 `空格` |
| `1`–`4` | 快捷选塔 |
| `Q` | 速度 ×1 / ×2 / ×3 |
| `Esc` | 取消选塔 / 暂停 |

## 防御塔

| 塔 | 费用 | 特点 |
|----|------|------|
| 🏹 箭塔 | 50 | 均衡单体 |
| 💣 炮塔 | 90 | 范围爆炸 |
| ❄️ 冰塔 | 75 | 减速 |
| ⚡ 激光塔 | 120 | 高伤射线 |

## 关卡（共 20）

从「新手村道」到「终焉之门」，难度、波次数与敌人强度逐步提升。
